Devin Flaky-Test Autofix

An independent verification layer for autonomous code remediation. Devin fixes flaky tests; before any PR reaches a human, this system independently re-runs the tests across many orderings and scans for cheat-fixes.

Why this exists

"CI is green" is not the same as "correct"

Coding agents made generating fixes cheap. They did not make trusting them cheap. The bottleneck moved: a human still has to verify every agent PR — and the signals we lean on (CI passed, the agent's own summary) are exactly the ones an agent optimizing "make CI green" can satisfy without solving the problem.

Flaky tests are the sharpest proof: a flaky test passes in CI's default order and fails under a different one, so CI-green provably lies. And the cheapest way for an agent to make one "pass" is to hide it — @pytest.mark.flaky, skip, a retry. CI goes green; the bug is now invisible and the test no longer protects you.

So this system treats "Devin opened a PR" as the start of verification, not the finish line.

The architecture

A single agent, wrapped in a deterministic verifier

One Devin agent does the engineering (per Cognition's own "don't build multi-agents": single-threaded, full context). The trust layer around it is deterministic — execution and static analysis, not another LLM judge that could share the agent's blind spots.

In practice

How a single remediation runs

01

Detect by re-ordering

Run the suite under many random seed orderings (pytest-randomly). A test that passes sometimes and fails sometimes is flaky because an earlier test leaked shared state.

02

Reproduce & bisect

Devin reproduces under the known-bad seeds, then bisects to the specific predecessor that leaks the state, and fixes the isolation at its root.

03

Verify independently

From a fresh clone, re-run the targets across the bad seeds + fresh seeds, scan the diff for skip/@flaky/retry, and confirm only test code changed.

04

Route on the verdict

Stable → ready for review. Cheat or still-flaky → the exact evidence goes back to Devin and it retries. Product bug → escalate, never mask.
